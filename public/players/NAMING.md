# Player photos

Drop headshots in this folder named `{slug}.jpg` (lowercase, hyphens). The
site picks them up automatically; players without a photo show their
initials in the same-size frame, so missing photos never break the layout.

The slug is the player's name lowercased with spaces → hyphens and
punctuation dropped — the same slug as the player's profile URL
(`/#/player/<slug>`). Current roster:

| Kingston Pace  | `kingston-pace.jpg` |

Tips:
- JPG only (`.jpg`, not `.jpeg`/`.png`) — the site only looks for `.jpg`.
- Square-ish crops look best; the frame is a circle. ~400×400 is plenty.
- Keep files under ~200 KB so pages stay fast.
